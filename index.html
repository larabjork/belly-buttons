Did you know...

Your body is an ecosystem of its own, with many bacteria living on you and inside you? Your belly button is no exception.


This research project gathers data from volunteers about the bacteria found in their belly buttons. You can select a test subject ID number at the left to see that anonymous individual's data.

What's an OTU?

OTU stands for "operational taxonomic unit". In this study, we're using an OTU ID number as shorthand for the scientific name of each bacterium. Some of those scientific names can get quite long, and OTUs are a quick way to refer to them.
